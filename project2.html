Project Overview

This project showcases the security tools I have used and the ethical hacking labs I have completed, focusing on vulnerability assessments, penetration testing, and exploitation techniques.

Phases of Ethical Hacking

The tools used in each phase of ethical hacking:

  • Reconnaissance: SHODAN, NMAP
  • Scanning: Nessus, NMAP
  • Gaining Access: SQL injection, Metasploit, Zphisher
  • Maintaining Access: Mimikatz
  • Covering Tracks: Ettercap, MD5 Collision

Tools and Labs Used

Nessus

Nessus is a vulnerability scanning tool used to identify security holes in software, systems, and networks. It's especially helpful in scanning for configuration errors, default passwords, and missing patches. Nessus is widely used during the scanning phase of ethical hacking to uncover weaknesses that can be exploited.

SHODAN

SHODAN is a search engine for Internet-connected devices. It helps ethical hackers in the reconnaissance phase by providing information about exposed devices such as webcams, routers, and servers, making it an excellent tool for network discovery.

NMAP

NMAP (Network Mapper) is a network discovery and security auditing tool. It is used during both the reconnaissance and scanning phases of ethical hacking to map out networks, identify active hosts, and find open ports. It's a critical tool in discovering potential vulnerabilities in the network.

Zphisher

Zphisher is a social engineering tool used to create phishing websites that look similar to popular websites (such as Facebook or Google) to trick users into giving up their credentials. This tool is particularly useful in the gaining access phase by simulating phishing attacks.

Metasploit

Metasploit is an exploitation framework used to find, exploit, and validate vulnerabilities. It is often used in the gaining access phase to exploit system weaknesses and test for security vulnerabilities.

SQL Injection

SQL injection is a technique used to attack data-driven applications by inserting malicious SQL queries. It is used to bypass authentication mechanisms and extract sensitive data from databases, and is part of the gaining access phase.

Mimikatz

Mimikatz is a post-exploitation tool used to extract plaintext passwords, PINs, and Kerberos tickets from memory. It is commonly used in the maintaining access phase to gain persistent control over a compromised system.

Ettercap

Ettercap is a tool used to perform Man-in-the-Middle (MITM) attacks. It allows hackers to intercept, capture, and manipulate data traveling between two systems, often used in the covering tracks phase after gaining access to the network.
